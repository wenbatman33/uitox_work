收入與付款說明

雲書在您同意下，將您的書籍上架到全世界的電子書平台銷售，同時彙整各平台的銷售紀錄，統整為您的銷售報表並付款給您。以下說明銷售收入計算及付款方式：

  • 業績查詢及報表

    您可於『業績查詢』查詢銷售狀況與收入。雲書接收到各電子書平台提供之銷售資訊時，業績報表即會更新。

    透過高效率的系統計算，雲書將您的檔案轉成ePub格式，您只需要簡單校稿，就可以免費擁有一本優質的電子書。

    雲書也與Google Play、iBooks等電子書平台合作，您可以透過雲書，將您的書銷售到全世界。

    例：iBooks於8/14提供8/10的銷售數字，您可於8/14或15在雲書網站查詢到您的業績資訊。

    ↑ 回到頂端頁面

  • 書籍銷售收入

    電子書銷售收入/本=書籍價格註*(1-電子書平台拆帳%)*(1-雲書服務費%)

    例1：若某書定價100元，在Google實際銷售價格為85元，則其銷售收入為：100*(1-48%)*(1-10%)=46.80元/本

    例2：若某書定價100元，在iBooks實際價格為85元，則其銷售收入為： 85*(1-30%)*(1-10%)=53.55元/本

    註：『書籍價格』在各電子書平台認定不同：

    ●Google Play的書籍價格=書籍定價，會依照作者的定價結算銷售業績

    ●iBooks的書籍價格為iBooks訂定，銷售收入之計算也以iBooks之實際價格為主

    例如：台幣定價100元的書，iBooks可能會用美金2.99作為書籍銷售價格

    ↑ 回到頂端頁面

  • 每月銷售收入

    每月銷售收入=該月電子書銷售總和 - 執行業務所得 - 二代健保 - 匯款手續費

    詳細名詞定義請參閱下方說明

    例1：若作者A書當月電子書銷售總和為新台幣$3,000，

    該月實際銷售收入= 3,000 - 30(匯款手續費) = 2,970


    例2：若作者B書當月電子書銷售總和為新台幣$6,500，

    該月實際銷售收入= 6,500 - 6,500*2%(二代健保) - 30(匯款手續費)= 6,340


    例3：若作者C書當月電子書銷售總和為新台幣$12,000，

    該月實際銷售收入= 12,000 - 12,000*10%(執行業務所得) - 12,000*2%(二代健保) - 30(匯款手續費) = 10,530

    ↑ 回到頂端頁面

  • 退書退款

    依照各電子書銷售平台規定，讀者有權於約定鑑賞期內退書，同時扣除書籍銷售金額。

    退書或扣款之資訊以各網站公告之銷售資訊為準。

    ●Google Play：讀者購買後七天內可退書

    ●iBooks：讀者購買後七天內可退書

    ↑ 回到頂端頁面

  • 電子書平台拆帳比例

    各電子書平台拆帳比例依照各平台規定為準。各平台撥款時將先扣除平台應得知銷售收入，再將餘款撥至雲書帳戶。

    各電子書平台之拆帳比例如下：

    ●Google Play：銷售業績之48%

    ●iBooks：銷售業績之30%

    ↑ 回到頂端頁面

  • 每月1日結算前月銷售收入

    雲書每月1日結算上月業績，依照各電子書平台公告之銷售銷售資訊為準。未統計之業績順延至下個月計算

    銷售收入將持續累計，累計至您第一次填寫收款銀行資訊後，才會開始撥款；之後每月1日自動結算匯款。

    ●例：作者A於10/31在iBooks 銷售一本書，11/02日iBooks通知雲書作者A在10/31有一本銷售。但雲書已經於11/01結算作者A 的10月業績，故此筆銷售將在計算11月業績時列統計，呈現在做者A 12/01結算之11月報表裏。

    ↑ 回到頂端頁面

  • 收款幣別

    業績報表呈現之幣別以您在『銀行帳號設定』時所約定之幣別為準，目前僅提供新台幣之服務。

    Google Play、iBooks目前均以美金付款給雲書。雲書會在每日收到電子書平台之銷售資訊時，將您的銷售收入換算成台幣呈現在銷售報表中。但是實際之付款金額會在付款日時才以當日匯率計算。

    詳見雲書付款方式之『匯率』。

    ↑ 回到頂端頁面

  • 收款銀行帳號

    您的收款人名稱需與收款銀行帳號完全相同，才可以撥款成功。

    您可以隨時修改您的領款銀行帳戶，將會於下一次付款日時生效。

    ↑ 回到頂端頁面

  • 地址、電話等資料

    依照稅法規定，銷售收入在一定金額以上，雲書將會為您代扣所得稅。

    請您詳實填寫，以作為報稅及退稅依據。

    ↑ 回到頂端頁面

  • 付款方式

    雲書每月25日直接將銷售收入匯款至您的指定銀行帳號，遇假日順延至下一工作日

    請確認您的收款收款銀行帳號 銀行帳號已經設定，第一次付款將在您第一次填寫領款帳號後匯款

    如因資料不全導致匯款失敗，將順延至次一付款日匯款

    實際入帳日期將視收款銀行作業為準

    ↑ 回到頂端頁面

  • 公司戶發票

    公司戶須先提供請款發票或收據後，才會撥款。

    雲書每月25日直接將銷售收入匯款至您的指定銀行帳號，遇假日順延至下一工作日。

    若您為國內公司，請與每月10日前，將發票正本掛號郵寄至台北市110信義區忠孝東路四段563號5F

    若您為國外公司，請於每月10日前將您的Invoice郵寄至payment@bestbook.com

    ↑ 回到頂端頁面

  • 幣別

    目前您的銷售收入以新台幣作為您的付款幣別。

    匯款時，新台幣金額將四捨五入到整位數。

    ↑ 回到頂端頁面

  • 匯率

    銷售報表顯示之銷售金額為估算值，實際匯率將於付款日前一天依照ORNDA顯示之匯率為準

    ●雲書依照國際網站ORNDA之匯率做為網站兌換匯率依據

    ↑ 回到頂端頁面

  • 實際付款金額

    實際付款金額與您的銷售收入的報表有差距，將扣掉政府規定須代扣之執行業務所得、二代健保、匯款手續費、以及雲書服務費，匯入您的指定帳戶。

    每次最低匯款金額需大於TWD 30元，若不足TWD 30元將遞延至下期付款

    ↑ 回到頂端頁面

  • 雲書服務費

    雲書只針對您有銷售成功的書籍，『扣除電子書平台費用後』，收取10%的服務費。服務費將在撥款給您時先扣除。

    ↑ 回到頂端頁面

  • 所得稅（執行業務所得）

    依所得稅法，銷售業績達一定金額以上，將代扣執行業務（稿費）所得之所得稅。

    若您為本國自然人，每次銷售收入大於新台幣20,000元以上時，才需要代扣10%執行業務所得之所得稅。

    執行業務所得一年有18萬的免稅額，若您年度結算時稿費或其他執行業務所得收入未超過18萬元，此項代扣稅額國稅局將會退還給您。

    針對中華民國境內之個人及營利事業之執行業務所得扣繳10%。如扣繳義務人每次應扣繳稅額不超過NTD2,000者，免予扣繳(各類所得扣繳率標準-第2條、第13條)

    針對非中華民國境內之個人及營利事業之執行業務所得扣繳20%。每次給付額不超過NTD5,000者，得免予扣繳。（各類所得扣繳率標準-第3條)

    ↑ 回到頂端頁面

  • 二代健保

    若您為本國自然人，每次銷售收入金額大於新台幣5000元以上時，雲書將會代扣2%二代健保費用。

    執行業務收入，應依規定之補充保險費率計收補充保險費，由扣費義務人於給付時扣取，並於給付日之次月底前繳納。目前最新公告補充保險費率為2%。（全民健康保險法第31、33條)

    ↑ 回到頂端頁面

  • 匯款手續費

    匯款手續費將於匯款時扣除。

    國內帳戶台幣匯款手續費：NTD30元/筆
