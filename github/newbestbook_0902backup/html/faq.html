常見問題

雲書讓您快速擁有高品質電子書(ePub)，您可以將檔案匯出、或是透過雲書銷售到全世界。 以下說明雲書網站的服務及功能，如果您還有任何問題，歡迎與我們聯絡。

  • 關於雲書

    雲書提供優質的ePub電子書。

    透過高效率的系統計算，雲書將您的檔案轉成ePub格式，您只需要簡單校稿，就可以免費擁有一本優質的電子書。

    雲書也與Google Play、iBooks等電子書平台合作，您可以透過雲書，將您的書銷售到全世界。

    ↑ 回到頂端頁面

  • 不需預付任何費用

    您不需要預先支付任何費用。

    只有當您的書透過雲書發佈到其他電子書平台、並且銷售成功後，雲書會從您的銷售收入中與您分享利潤。更多說明可見收入與付款說明。

    ↑ 回到頂端頁面

  • 電子書版權歸作者擁有

    透過雲書出書，電子書的版權歸作者（或合法代理人）所擁有，雲書經您授權同意後轉製並代為發佈銷售。

    ↑ 回到頂端頁面

  • 關於書的版權

    轉檔前請先確認您擁有該書版權，或是您已經被授權可以代理這本書，您才可以將此本書發佈到不同平台

    ↑ 回到頂端頁面

  • 支援檔案種類

    目前雲書支援Word、PDF、ePub三種檔案格式，上傳前請確認您的檔案沒有鎖密碼或是保護，才能順利轉檔成功。

    ↑ 回到頂端頁面

  • 轉檔結果及閱讀

    雲書目前的轉檔為epub3橫排的樣式，匯出後可在iBooks, Kindle, Google Play等可支援ePub的APP上流暢閱讀

    ↑ 回到頂端頁面

  • 支援檔案內容

    無論您的書是直排或是橫排，若您的書屬於純文字或圖文書，目前都會有非常好的轉檔效果，您只需要少許的時間校稿，檢查排版及可能的錯字，

    但若檔案內容較複雜，例如圖片多或多欄排版(如雜誌)，轉檔結果與原稿會有出入，您可以使用進階編輯功能校稿，讓版面符合您的預期效果。

    ↑ 回到頂端頁面

  • 轉檔失敗

    失敗原因可能有：1.您上傳的檔案有鎖密碼、2.檔案損毀、3.PDF為保護狀態（文字複製到其他檔案為亂碼）、4.其他。

    您可以參考『作品列表』右上方的『通知』（鈴鐺形狀圖示），有失敗原因說明。

    您也可以將檔案重新上傳，或是把檔案寄給我們，讓我們協助您轉檔。請與我們聯絡。

    ↑ 回到頂端頁面

  • 檔案重新上傳

    目前僅支援上傳失敗的檔案可以重新上傳。

    您可以至『作品列表』的書籍旁邊找到『重新上傳』的按鈕，或者也可以直接『新增書籍』，製作一本全新的書。

    ↑ 回到頂端頁面

  • 線上預覽

    雲書提供線上預覽，轉檔完後您可使用線上模擬器預覽閱讀效果，可選手機、平板、PC等不同尺寸的閱讀器、用直式或橫式預覽轉檔完的電子書。

    預覽結果可能與實際閱讀時可能有些許落差，原因是每個讀者使用的APP不同，或是各閱讀器（手機或平板）的解析度不同。

    ↑ 回到頂端頁面

  • 目錄與內容修改

    我們提供進階編輯的功能，您可以新增、刪除文字、表格、圖片等所有電子書內容及目錄。

    ↑ 回到頂端頁面

  • 轉檔比對與建議微調

    若您上傳的檔案為PDF檔，雲書在『目錄及內容修改』還提供更多進階編輯功能：

    1.『轉檔比對』—您的原始檔與電子書左右直接比對，方便校稿。

    2.『建議微調』—系統建議您直接檢查需要仔細校稿的地方，節省您全文校稿的時間。

    ↑ 回到頂端頁面

  • 免費匯出及使用

    您目前可以將轉好的電子書檔案(ePub)免費匯出（須先註冊成為雲書會員）。

    若您擁有該書的版權，匯出的電子書檔案也可供您合理使用。

    ↑ 回到頂端頁面

  • 標題、作者、語系、分類、書籍介紹

    以上內容為必填欄位，填寫完後方能上架至各電子書平台。填寫越多資訊，可以讓更多讀者容易找到您的書。

    您可隨時修改以上內容，待平台審核後自動更新上架資訊。

    ↑ 回到頂端頁面

  • 副標、出版日期、關鍵字等

    以上內容為選填，不會影響書籍上架與否。您填寫的資料越多，可以更幫助讀者瞭解您的書，也會更增加購買意願。

    您可隨時修改以上內容，待平台審核後自動更新上架資訊。

    ↑ 回到頂端頁面

  • 標題、作者、語系、分類、書籍介紹

    雲書提供封面設定的功能。您可以透過雲書上傳您編輯好的封面， 或是選擇雲書提供的系統圖庫編輯標題文字，為您的電子書選擇一個合適的封面。

    ↑ 回到頂端頁面

  • 銷售平台選擇

    您可以透過雲書，將書籍發送到全世界的電子書平台銷售。

    雲書目前支援Google Play及iBooks，您可以透過雲書將書籍發佈到這兩個網站。

    您也可以選擇不發佈到任何平台，只透過雲書『連結分享』的功能，將您的電子書分享給您的讀者。

    ↑ 回到頂端頁面

  • 上架規則

    各電子書平台都會先審核您的電子書內容及書籍介紹後，再決定是否上架。 各電子書平台的相關規定略有不同，請於上架前勾選平台時注意告電子書平台規定。

    上架銷售前，請確認您擁有該書的版權或代理權，才可以合法發佈銷售發佈該本電子書。

    ↑ 回到頂端頁面

  • 定價設定

    您可以自訂您希望的銷售價格。各電子書平台會參考您的售價，重新訂定該平台的銷售價格。實際售價及幣別將以各平台網頁顯示為準。

    ↑ 回到頂端頁面

  • 免費書

    您可以設定您的電子書為0元，讓更多讀者可以看到您的書。

    免費書在雲書及各電子書平台都可以全文閱讀，您仍可以隨時修改設定。

    ↑ 回到頂端頁面

  • 價格修改

    發佈銷售後，您仍可以隨時修改您的價格。待各電子書平台審核後，系統即會自動更新您的資料。

    ↑ 回到頂端頁面

  • 刪除

    透過刪除功能，您可以將已上架到各電子書平台的電子書停止銷售。

    但是為保障購買過的讀者權益，各電子書平台允許買過本書的讀者再次下載本書。

  • 下架

    若您的書有被檢舉或違反法規規定，各電子書平台有權將您的書下架。

    ↑ 回到頂端頁面

  • 分享連結

    書籍發佈成功後，您可以透過URL分享、Facebook、Twitter等將此本電子書分享給您的讀者，讓更多人看見您的書。

    ↑ 回到頂端頁面

  • 關於試閱

    試閱範圍視各電子書平台規定而有所不同，如果您的書籍售價設定大於0元（非免費書），各平台的試閱規則如下：

    ・雲書：書籍的10%

    ・Google Play：書籍的20%

    ・iBooks：書籍的10%

    ↑ 回到頂端頁面
